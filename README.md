# PasswordStrength-Checker
A interactive password strength checker built with HTML, CSS, and JavaScript. This project provides a user-friendly interface to check the strength of a password, providing real-time feedback on password complexity.

Features:
Checks for uppercase, lowercase, numbers, and special characters
Provides password strength indicators: Weak, Medium, and Strong
Displays password strength message and updates in real-time
Responsive design for smaller screens

Technologies Used:
HTML5
CSS3 (with Google Fonts and media queries)
JavaScript (with DOM manipulation and event listeners)

How it Works:
The user inputs a password into the password field.
The JavaScript code checks the password complexity using regular expressions.
Based on the password complexity, the code updates the password strength message and indicator.
The password strength message and indicator are updated in real-time as the user types.
